Objetivo do Projeto

O objetivo principal deste trabalho é criar uma galeria de imagens online que seja visualmente agradável, responsiva e performática. O site foi desenvolvido como projeto final da disciplina, demonstrando a aplicação prática de tecnologias web modernas para construir uma interface de usuário funcional e elegante, focada na experiência do usuário.

Recursos e Tecnologias

  • HTML5 Semântico: Para uma estrutura bem organizada e acessível.
  • CSS3 Moderno: Utilizamos Flexbox para o layout da navegação e CSS Grid para criar a galeria de imagens responsiva, garantindo uma adaptação perfeita a qualquer tamanho de tela.
  • Design Responsivo (Mobile-First): A estilização foi pensada primeiro para dispositivos móveis, garantindo que a experiência seja excelente em telas pequenas, e depois aprimorada para desktops.
  • Git e GitHub: Para controle de versão do código, trabalho em equipe através de branches e gerenciamento de Pull Requests.

Próximos Passos

  • Implementação de um formulário de contato funcional.
  • Adição de um Lightbox para visualizar as imagens da galeria em tela cheia.
  • Otimização de performance, como o carregamento tardio (lazy loading) das imagens.
  • Criação de mais páginas de conteúdo.
